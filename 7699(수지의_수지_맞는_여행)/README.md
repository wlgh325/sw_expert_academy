# 문제정리
1. 섬은 R행 C열로 이루어져있다.
2. 알파벳은 섬의 명물이고 같은 알파벳을 가지면 같은 명물을 가진다.
3. 명물을 볼때 마다 요금을 지급하며, 처음 볼때는 무료이다.
4. 명물을 본 후 4방향(상,하,좌,우)중 한 방향으로 이동 가능하다.
5. 같은 알파벳 명물을 두 번 이상 보지 않도록 여행을 떠나는 방법 중, 볼 수 있는 최대 명물의 개수를 구해라
