# 문제정리
1. 연산자와 숫자가 주어질때 가능한 수식을 계산하여 최대와 최소값을 구해라.
2. 연산자의 우선순위는 고려하지 않고 왼쪽에서 오른쪽으로 차례대로 계산한다.
3. 연산자의 개수는 숫자의 개수보다 항상 1개 작다.
4. 숫자의 순서는 바꿀 수 없다.
5. 나눗셈에서 소수점 이하는 버린다.
6. 수식을 완성할때 주어진 카드를 모두 사용해야 한다.

그리고 최대 최소를 구하기 위해서 Collections.min과 max 함수를 이용합니다.
